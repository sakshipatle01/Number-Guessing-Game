# 🎮 Funny Number Guessing Game (Java Console Project)

A fun and beginner-friendly Java console game where the computer selects a random number and the user has to guess it within limited attempts. The game gives funny hints instead of direct answers 😄

---

## 🚀 Features

- Random number generated between 1 to 10
- 5 attempts to guess the correct number
- Funny hints after every wrong guess
- Hint system:
  - 📈 Too Big → try smaller number
  - 📉 Too Small → try bigger number
- Win/Lose messages
- Beginner-friendly Java code

---

## 🛠️ Technologies Used

- Java (Core Java)
- Scanner Class (User Input)
- Random Class (Number Generation)
- Loops (for attempts)
- Conditional Statements (if-else)

## ▶️ How to Run

Step 1: Compile the program  
javac GuessGame.java  

Step 2: Run the program  
java GuessGame  

---

## 🎯 How to Play

1. Computer selects a random number between 1 to 10  
2. You get 5 chances  
3. Enter your guess each time  
4. After each wrong guess:  
   - 📈 If guess is too high → "Too Big bro 😆"  
   - 📉 If guess is too low → "Too Small bro 😜"  
5. If correct → 🎉 You win  
6. If not → 💀 Game Over and correct number is shown
